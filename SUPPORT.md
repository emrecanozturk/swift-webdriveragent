# Support

Use GitHub Issues for:

- Build failures.
- Endpoint compatibility gaps.
- Documentation fixes.
- Runtime smoke-test results.

Include:

- Xcode version.
- iOS version.
- Device model.
- Whether the failure is build, signing, launch, transport, or endpoint behavior.
- Sanitized command output.

Do not include certificates, profiles, UDIDs, private bundle ids, or internal hostnames.
