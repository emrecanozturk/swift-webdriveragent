# Roadmap

## Near Term

- Validate real-device launch across the supported iOS/Xcode matrix.
- Expand endpoint compatibility tests.
- Add more explicit client examples for Appium-style attachment.
- Publish the GitHub Wiki from `docs/wiki`.

## Medium Term

- Improve source-tree performance for large accessibility hierarchies.
- Add structured benchmark output for screenshot, source, and gesture latency.
- Add more compatibility fixtures for common mobile automation clients.
- Improve native location simulation diagnostics by iOS version.

## Non Goals

- Reimplement every upstream WebDriverAgent private integration.
- Include signing assets or provisioning profiles.
- Replace MDM, supervision, or device-farm scheduling systems.
- Expose the runner directly to untrusted networks.
